Data download

You can download the xml file corresponding to this planetary system, which is part of the Open Exoplanet Catalogue. All information on this page has been directly generated from this XML file. You can also download the entire catalogue over at github. If you prefer to download the dataset as an ASCII tables, you might find the oec_tables repository useful.

Corrections

If you spot an error or if you can contribute additional data to this entry, please send an e-mail to exoplanet@hanno-rein.de. Please include the corrected xml file and a reference to where the new data is coming from, ideally a scientific paper. If you are fluent with git and github, you can also create a pull request or open an issue on the Open Exoplanet Catalogue repository. Please include the reference to the relevant scientific paper in your commit message. {% endblock content %}
